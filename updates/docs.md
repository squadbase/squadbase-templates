# Template Updates

## description.md について

- 新機能・機能改善: ✨
- バグ修正: 🐛
- セキュリティアップデート: 🔒
- その他のアップデート: ♻️

とします。フォーマットは以下

```markdown
**✨ Title of the update**

Description of the update.
```

- 英語で書いてください。
- 非デベロッパーにわかりやすい表現にすること。

## instruction.md について

- エージェントが確実にアップデートを遂行できるように、指示を明確にかつ具体的に書いてください。ステップバイステップの形式や実装するべきコードなどを含めてください。解釈の余地を与えないようにしてください。
- 英語で書いてください。
- h1(`#`)は使えません。ヘディングの最大レベルは h2(`##`)にしてください。
