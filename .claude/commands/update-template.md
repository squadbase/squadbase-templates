# テンプレートのアップデート

指定されたアップデート指示ファイルに基づいて、テンプレートを更新します。

## 利用方法

Format: `/update-template <template-path> <update-instructions-path>`

[Example] Update Core(nextjs-dashboard-starter) Template: `/update-template core updates/nextjs-dashboard-starter/v1-to-v2/instruction.md`

[Example] Update BigQuery(bigquery-starter) Template: `/update-template bigquery updates/bigquery-starter/v1-to-v2/instruction.md`
