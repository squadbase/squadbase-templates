# アップデートファイルを作成する

## 利用方法

Format:

```text
/create-update <template-path> <user-update-instructions>
```

[Example] Update Core Template:

```text
/create-update core サイドバーの改善をしたいです。現在開いているページの項目がハイライトされるようにしてください。
```

## 指示

まず、@updates 以下の既存のファイルと@updates/docs.md を読んで、テンプレートのアップデートについて理解してください。次に、対象のテンプレートプロジェクトとユーザーのアップデート指示に基づいて、以下の 2 つのファイルを作成してください。

- instruction.md
- description.md
